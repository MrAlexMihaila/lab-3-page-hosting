Some of My Projects

This project was an attempt to make a "full" game in a month. Despite the tight deadline, I was able to make a platformer with five levels where you needed to collect all of an item in the level before you're able to move on to the next level. The final level also featured a "boss", if you can even call it that. On top of the deadline, I also made an effort to make a system that would allow porting and testing on mobile devices to be as easy as possible.

This project is a mod for Minecraft to allow the client to communicate with Discord in order to display information in the account's status, such as how long they have been playing for and what item they are holding. This "project" deserves mention since it has been downloaded/used over 80 thousand times as of 1/29/2026. This project made me more confident in my skills as a programmer and designer.
